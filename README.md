## how to make

1. Wordファイルをテキストに [docx2txt](https://github.com/code4fukui/docx2txt)
2. 各種画像をサイズ調整JPEGに [tojpg-mac](https://github.com/code4fukui/tojpg-mac)
3. 変換済みデータをCSVに [data2csv.js](data2csv.js)
