# sabaejazzfes2025

鯖江JAZZフェスティバル2025の飲食出店者情報を紹介するウェブサイトです。HTML、CSS、およびデータ処理用のDenoスクリプトで構築されています。

## デモ

- **[ライブサイト](https://code4fukui.github.io/sabaejazzfes2025/)**: 出店者情報をカード形式で表示するレスポンシブなビュー。
- **[データビューアー](https://code4fukui.github.io/sabaejazzfes2025/list.html)**: 生データのシンプルな表形式ビュー。

## 機能

- **動的な出店者リスト**: 出店者情報はCSVファイルから直接読み込まれます。
- **レスポンシブデザイン**: デスクトップからモバイルまで、さまざまな画面サイズに適応します。
- **データ自動化**: Denoスクリプト（`data2csv.js`）により、テキストおよび画像ファイルが配置された構造化ディレクトリから `shops.csv` データファイルを自動生成します。
- **オープンデータ**: すべての出店者情報はオープンデータとして提供されます。

## データソース

出店者リストはCC BYライセンスの下でオープンデータとして提供されます。

- **データファイル**: [`shops.csv`](shops.csv)
- **クレジット**: [鯖江JAZZフェスティバル](https://sabaejazz.jp/)

## データの更新方法

ウェブサイトのコンテンツは、ローカルの `data/` ディレクトリに配置されたファイルから生成されます。出店者情報を追加または更新するには、以下の手順に従ってください。

### 前提条件

- データ処理スクリプトを実行するには、[Deno](https://deno.land/) がインストールされている必要があります。
- ソースファイルの準備には以下のツールの使用を推奨します:
  - [docx2txt](https://github.com/code4fukui/docx2txt): 出店者の説明文を `.docx` から `.txt` に変換します。
  - [tojpg-mac](https://github.com/code4fukui/tojpg-mac): 画像をリサイズし、JPEG形式に変換します。

### 手順

1.  **データディレクトリの準備**:
    プロジェクトルートにローカルの `data/` ディレクトリを作成します（このディレクトリはGitの管理対象外です）。以下の構造で出店者情報を整理してください:

    ```
    data/
    └── [カテゴリ]/
        └── [出店者名]/
            ├── description.txt
            ├── image1.jpg
            └── image2.png
    ```

    - `[カテゴリ]`: 出店者の種類（例: `飲食ブース`、`クラフトビール`）。
    - `[出店者名]`: 店舗またはブースの名前。
    - 出店者の説明文をプレーンテキストファイル（`.txt`）として配置します。
    - 1つ以上の画像（`.jpg`、`.jpeg`、`.png`）を同じフォルダに配置します。

2.  **処理スクリプトの実行**:
    プロジェクトルートからDenoスクリプトを実行します。このスクリプトは `data/` ディレクトリのデータを読み取り、`shops.csv` を生成して、画像を `img/` ディレクトリにコピーします。

    ```bash
    deno run -A data2csv.js
    ```

3.  **変更のコミット**:
    スクリプトが正常に実行された後、更新された `shops.csv` ファイルと `img/` ディレクトリ内の新しい画像をリポジトリにコミットしてください。
